# 🎮 Tic Tac Toe - Modern React App

A visually stunning, neon-themed Tic Tac Toe web game built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**. It features solo play against an intelligent AI or local multiplayer with friends, complete with a clean and interactive interface.

---

## ✨ Features

- **👾 Play vs Computer:** Face off against the computer across three tiers of difficulty:
  - 🌱 **Easy:** Great for beginners!
  - ⚖️ **Medium:** A bit of a challenge.
  - 🔥 **Hard:** Unbeatable AI using the Minimax algorithm!
- **👥 Local Multiplayer:** Play side-by-side with a friend on the same device.
- **🏷️ Custom Player Names:** Set custom player names for X and O natively before the match begins. Let your friends know who is winning!
- **🏆 Live Scoreboard:** Dynamic scoreboard tracking wins for Player X, wins for Player O, and Draws.
- **🎨 Glassmorphic & Neon Aesthetics:** Beautifully crafted dark mode UI with glow effects, sleek animations, blurred backdrops, and interactive hovering effects mapping tailwind utilities.
- **📱 Responsive:** The layout maps perfectly to mobile screens as well as wide desktops natively.

## 🚀 Getting Started

Follow these steps to get a local copy of this project up and running.

### Prerequisites

Make sure you have Node.js installed on your computer.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kehad/XO-Game.git
   ```

2. Navigate to the project directory:
   ```bash
   cd xo-game
   ```

3. Install all the necessary dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to the localhost link provided in the console (usually `http://localhost:5173/`).

## 📜 How to Play

1. **Choose a Game Mode:** On the main menu, pick whether to play locally with a friend (👥 Local Multiplayer) or against the AI (🤖 Play vs Computer).
2. **Enter Optional Names:** Type your name and your opponent's name into the input boxes provided. The game will reflect these inputs on the scoreboard!
3. **Select Difficulty (If vs Computer):** If playing against the computer, select between Easy, Medium, or Hard difficulty. High-tier modes use Minimax to prevent you from ever scoring a win.
4. **Make Your Moves:** Click the tiles. The first player to align 3 consecutive marks vertically, horizontally, or diagonally wins!
